# Portfolio Website Development and Deployment Plan

## Objective
Create a modern, dark-themed portfolio website for a data analyst ("A Puchong") and deploy it via GitHub Pages.

## Tech Stack
*   **HTML5, CSS3, Vanilla JavaScript**
*   **Font:** JetBrains Mono (via Google Fonts)

## Implementation Steps

1.  **Project Setup**
    *   Create local directory with: `index.html`, `style.css`, `script.js`, `assets/`.
    *   Initialize a local Git repository.

2.  **Frontend Development**
    *   **HTML Structure**: Define sections (Hero, Projects, Contact).
    *   **Styling**: Implement a modern dark theme using CSS variables and apply the JetBrains Mono font.
    *   **Content**:
        *   **Hero**: Name "A Puchong" and subtitle "Data Analyst".
        *   **Projects**: Placeholder cards for 3 projects.
        *   **Contact**: Add relevant contact placeholders (email, LinkedIn, GitHub).
    *   **Interactivity**: Add basic JavaScript for smooth scrolling or simple UI interactions.

3.  **Version Control & Deployment**
    *   Commit changes locally.
    *   Create a repository on GitHub and push the `main` branch.
    *   Configure **GitHub Pages** in the repository settings to deploy from the `main` branch.

#### **Verification & Testing**
*   **Visual Check**: Confirm the dark theme and font rendering.
*   **Responsiveness**: Verify layout on mobile/desktop browsers.
*   **Deployment**: Ensure GitHub Actions successfully deploys the site.
