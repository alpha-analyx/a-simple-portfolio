# A Puchong | Data Analyst Portfolio

A modern, responsive portfolio website for a Data Analyst, featuring a dark-themed aesthetic and a built-in light/dark mode toggle.

## 🚀 Features

- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Theme Switcher:** Toggle between dark and light modes with a smooth transition.
- **Data-Focused Aesthetic:** Styled with the "JetBrains Mono" font for a clean, technical look.
- **Project Showcase:** A grid-based layout to highlight key data analysis projects.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables) and Grid layout.
- **JavaScript:** Vanilla JS for theme switching functionality.
- **Fonts:** [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts.

## 📂 Project Structure

```text
docs/
├── index.html    # Main HTML structure
├── style.css     # Styling and theme variables
└── script.js     # Interactivity and theme logic
README.md         # Project documentation
```

## 🌐 Deployment

This project is designed to be hosted on **GitHub Pages** using the `docs` folder. To deploy:
1. Push your code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose the `main` branch and the `/docs` folder, then click **Save**.
